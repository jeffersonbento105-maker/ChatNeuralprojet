Objetivo: Corrigir a funcionalidade do balão de apresentação do Chat Neural para que ele *troque dinamicamente entre Clark e Ragnaria* quando o usuário clicar no botão de troca de personagens. *Nenhum design, layout, dimensão ou outra funcionalidade pode ser alterada.*

Instruções detalhadas para a IA:

1. *Função principal*
   - O balão de apresentação deve exibir o *nome correto do personagem* selecionado pelo usuário (Clark ou Ragnaria).  
   - *Não alterar CSS, HTML, cores, fontes, tamanho, borda, layout ou qualquer outro aspecto visual*.  
   - Apenas o *texto dentro do balão* pode ser modificado.

2. *Personagens e textos*
   - Clark:  
     "Olá, eu sou o Clark, sou assistente de IA, analítica, posso ajudar você com explicações detalhadas, soluções passo a passo e respostas estruturadas, como posso ajudar hoje?"
   - Ragnaria:  
     "Olá, eu sou o Ragnaria, sou assistente virtual, IA, analítica, posso ajudar você com explicações detalhadas, soluções passo a passo e respostas estruturadas, como posso ajudar hoje?"

3. *Estrutura dos arquivos*
   - index.html → contém o balão (div) e o botão de troca (button) com IDs claros.  
   - script.js → *somente adicionar ou ajustar a lógica de troca do texto do balão*.  
   - style.css → *não deve ser alterado*.

4. *Lógica JS*
   - Criar ou complementar um *listener de clique* no botão de troca.
   - Verificar o personagem ativo e atualizar apenas o texto do balão conforme o personagem.
   - Nenhum outro elemento ou funcionalidade pode ser modificado.
   - A troca deve ocorrer *instantaneamente*, sem recarregar a página.

5. *Exemplo de código de referência para a IA*
```javascript
const toggleButton = document.getElementById('toggleCharacter');
const balloon = document.getElementById('introBalloon');

toggleButton.addEventListener('click', () => {
  if(toggleButton.dataset.character === 'Clark'){
    balloon.innerText = "Olá, eu sou o Ragnaria, sou assistente virtual, IA, analítica, posso ajudar você com explicações detalhadas, soluções passo a passo e respostas estruturadas, como posso ajudar hoje?";
    toggleButton.dataset.character = 'Ragnaria';
  } else {
    balloon.innerText = "Olá, eu sou o Clark, sou assistente de IA, analítica, posso ajudar você com explicações detalhadas, soluções passo a passo e respostas estruturadas, como posso ajudar hoje?";
    toggleButton.dataset.character = 'Clark';
  }
});
