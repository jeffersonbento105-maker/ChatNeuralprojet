Tarefa: adicionar um tooltip em CSS (sem JS) ao meu botão de Send Email, mantendo exatamente o mesmo tamanho e a mesma posição (canto superior direito).
Regras importantes:

Não altere width, height, padding, margin ou font-size do botão já existente.

Mantenha o botão no canto superior direito (top/right). Se ele já estiver posicionado, não mude os valores; se não estiver, aplique position: fixed; top: 15px; right: 15px;.

Adicione apenas um atributo data-tooltip com o texto:
With Chat Neural, you can create formal, friendly, and neutral emails within the chat

O tooltip deve aparecer no hover do botão, logo abaixo dele, com leve transição de opacidade.

O código deve ser HTML + CSS puro (sem JS).

Use o id send-email-btn no botão. Se meu botão já existir, apenas adicione o id e o atributo data-tooltip, sem mudar o restante.

Se o botão ainda não existir, crie-o com um ícone de email em SVG, mas não defina largura/altura (para não mudar o tamanho).

Código (HTML + CSS)

Cole o CSS dentro de <style> e garanta que o botão tenha id="send-email-btn" e o data-tooltip com a frase.
